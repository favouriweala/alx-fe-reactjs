# Recipe Sharing App

A simple Recipe Sharing Application built with **React** and **Zustand** for state management.  
This project allows users to add, view, and manage recipes in a clean and responsive interface.

## **Features**

- Add a new recipe with a title and description.
- View a list of all recipes.
- State management with Zustand for fast and efficient updates.
- Responsive design with Tailwind CSS (optional for styling).

---

## **Technologies Used**

- React 18
- Zustand
- Tailwind CSS (optional)
- Vite (for development and build)
- JavaScript (ES6+)
